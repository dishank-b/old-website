Bachelor's Thesis Project, IIT Kharagpur

|
Actor Critic Algorithm
Implementation of Actor Critic Network
Training on TORCS

Here is the the complete report of the work.

This work explores the use of deep reinforcement learning for end-to-end control of an autonomous driving car. The driving module is trained using the actor-critic algorithm. Ornstein-Uhlenbeck (OU) process is used for exploration. Here the actor is implemented using a neural network which predicts future actions given the inputs state values and the critic is implemented using another neural network which criticizes the actions of the actor. The model is trained using TORCS game engine on road tracks with no traffic conditions for about 14 hours from scratch. The agent initially struggles during first two hours of training and eventually learns to drive without crashing. The performance of agent is observed to saturate after 12 hours of training.
Adding to the work done, currently I have been working on a research problem of my own for next part of Bachelor’s Thesis Project. I believe that successful use of self driving cars is only possible when all the cars in the traffic will be self-controlled and no manual control is allowed. For this, I have been trying to model driving for multiple vehicles in the traffic, using multi-agent reinforcement learning and game theory. Game theory will be used to dictate the terms of traffic as collabarative game whereas reinforcement learning will be used to decide policy for an individual car. I am taking only raw camera frames as an input which will make this problem very similar to real life scenarios. Hence, solution to this problem will give driving policy for individual vehicle in traffic given their camera feed as input. I have established the basic model for formulating the traffic and individual policy networks, further work for solving techniques is yet to be done.
